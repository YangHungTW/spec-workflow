Command palette design decisions:
1. Palette is context-sensitive: it pre-populates the focused session slug from the last-viewed card.

4. "Advance to next stage" is the first ranked item when a stalled session is the context — surfacing the most urgent action at the top.

Context menu design decisions:
1. First two items are always the primary control actions for the session's current state (Advance + Message when stalled; no advance button when active).
2. Specflow commands are shown inline with their command slug — no need to open command palette for common operations.
3. Session actions (Open in Finder, Copy Path) are separated by a divider — these are read-only and never trigger confirmation.
4. Destructive commands are at the bottom, danger-styled, with the DESTROY pill, and end in "…" to signal a confirmation step follows.
