The PollingFooter is fully removed. The sidebar pip is the only liveness indicator. No polling interval is surfaced to the user — the graph simply updates when files change.
Design decision — live-update affordance: Chose a single sidebar pip + per-node timestamp whisker over (a) keeping the PollingFooter or (b) a floating "last refreshed" banner. Rationale: the sidebar pip answers "is the watcher alive?" at a glance; the per-node whisker answers "when did this artifact last change?" — the two pieces of information the user actually needs. A global "last refreshed" timestamp is meaningless when only one node in the graph changed.
