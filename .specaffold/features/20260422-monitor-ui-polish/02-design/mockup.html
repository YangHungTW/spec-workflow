Item 3 — Stage Tabs Disabled When Artefact Missing

CHANGED

The TabStrip already has a .tab-strip__tab--missing CSS class with opacity:0.5; cursor:not-allowed and a tooltip via title. This item refines that treatment: the disabled opacity is lowered to 0.38 (more obviously non-interactive), and the tooltip is now rendered as a CSS ::after tooltip so it is visible on hover without relying on the OS native title attribute popup (which has a delay and inconsistent appearance across platforms).
