- [git cat-file --batch for staged-file scan](git-cat-file-batch-for-staged-file-scan.md) — Use `git cat-file --batch` single persistent process when scanning N staged files in a hook-budget path; per-file `git show :FILE` forks N times and busts the 200ms hook latency budget.

- [Env-var file path boundary check](envvar-file-path-boundary-check.md) — Test scripts accepting env-var file path overrides must canonicalise via `cd dirname && pwd -P` and assert path is under `REPO_ROOT`; use `${var#$REPO_ROOT/}` strip test (not `case` inside subshell); for executables also assert `[ -x ]`. Reviewers block missing checks as security must.
